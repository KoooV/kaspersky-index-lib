# Текстовый индексатор

Простая библиотека для индексации текстовых файлов и поиска слов в них.

## Описание

Эта библиотека позволяет добавлять файлы и директории в индекс, а затем выполнять быстрый поиск по словам в проиндексированных файлах. Приложение также отслеживает изменения в добавленных директориях и автоматически обновляет индекс.

## Требования

Для сборки и запуска проекта вам понадобятся:
- **Java Development Kit (JDK)**: версия 21 или выше.
- **Apache Maven**: версия 3.6 или выше.

## Настройка окружения

1.  **Клонируйте репозиторий**:
    ```bash
    git clone <URL_вашего_репозитория>
    cd text-indexer-lib
    ```

2.  **Убедитесь, что переменная `JAVA_HOME`** указывает на директорию с JDK 21+, и `mvn` доступен в вашем `PATH`.

## Сборка проекта

Для сборки библиотеки и создания исполняемого `.jar`-файла выполните следующую команду в корневой директории проекта:

```bash
mvn clean install
```
Эта команда скомпилирует исходный код, запустит тесты и упакует проект в `target/text-indexer-lib-1.0-SNAPSHOT.jar`.

## Запуск приложения

После успешной сборки вы можете запустить консольное приложение:

```bash
java -jar target/text-indexer-lib-1.0-SNAPSHOT.jar
```

Приложение запустится в интерактивном режиме. Доступные команды:

-   `add <путь>`: Добавить файл или директорию в индекс для отслеживания.
    -   *Пример*: `add /home/user/documents`
-   `search <слово>`: Найти все файлы, в которых содержится указанное слово.
    -   *Пример*: `search hello`
-   `exit`: Завершить работу приложения.

## Запуск тестов

Для запуска модульных тестов выполните команду:

```bash
mvn test
```

Результаты тестов будут отображены в консоли.
